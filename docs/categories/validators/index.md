# Validators

Schema validation and business rule checking tools for mobility data.

## Overview

Validators help ensure your mobility data meets standard requirements and business rules. These tools can:

- Validate XML/JSON against schemas
- Check business rule compliance
- Verify data quality
- Test standard conformance

## Available Tools

Currently, we don't have any validators listed in the catalog yet.

## Wanted: Validator Tools

We're looking for tools that can:

- **DATEX II Validators**: Validate DATEX II XML messages
- **NeTEx Validators**: Check NeTEx profile compliance
- **SIRI Validators**: Validate SIRI messages
- **mobilityDCAT-AP Validators**: Check metadata compliance
- **Multi-Standard Validators**: Support multiple mobility standards

## Why Validators Matter

Validators are crucial for:
- Ensuring interoperability
- Catching errors early
- Maintaining data quality
- Meeting standard requirements
- Reducing integration issues
