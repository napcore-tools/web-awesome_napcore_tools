# Metadata Tools

Catalog generation and discovery tools for mobility data metadata.

## Overview

Metadata tools help you create, manage, and discover information about mobility datasets. These tools support:

- Creating standardized metadata
- Generating data catalogs
- Publishing dataset descriptions
- Enabling data discovery
- Harmonizing metadata across sources

## About mobilityDCAT-AP

mobilityDCAT-AP is a metadata specification developed by NAPCORE to enhance cross-border and cross-sectorial discoverability of mobility-related datasets. It extends DCAT-AP (the European standard for data portal metadata) with mobility-specific elements.

**Key Resources:**
- **Specification**: [w3id.org/mobilitydcat-ap](https://w3id.org/mobilitydcat-ap)
- **GitHub**: [mobilityDCAT-AP organization](https://github.com/mobilityDCAT-AP)
- **Vocabulary**: [Controlled vocabularies](https://github.com/mobilityDCAT-AP/controlled-vocabularies)

## Planned Tools

We'd love to see more metadata tools in the catalog:

- **Metadata Validators**: Validate mobilityDCAT-AP compliance
- **Catalog Harvesters**: Collect metadata from multiple sources
- **Metadata Converters**: Transform between metadata formats
- **Quality Checkers**: Assess metadata completeness and quality
